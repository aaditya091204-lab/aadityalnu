# Web Development Roadmap Notes

## Backend Notes

**Programming Languages:** Python, Java, JavaScript, C, Go  
- Python: beginner-friendly, simple syntax  
- Java: solid, widely taught in CS  
- JavaScript: full-stack capable  
- C / Go: performance-focused  

**Version Control:** Git & GitHub – track code changes, collaborate  

**Data Structures & Algorithms:** Essential for problem solving and interviews  

**Design Patterns:** 23 classic patterns (Gang of Four), useful for frameworks like Django, Spring Boot, ASP.NET Core  

**Databases:**  
- Relational: MySQL, PostgreSQL, SQL Server  
- NoSQL: MongoDB, CouchDB  
- SQL for queries, MongoDB for flexible schemas  

**Frameworks:**  
- Python → Django  
- Java → Spring Boot  
- JavaScript → Express.js  
- C → ASP.NET Core  
- Ruby → Ruby on Rails  
- Go → Gin  

---

## Frontend Notes

**Core Languages:**  
- HTML: structure  
- CSS: styling  
- JavaScript: interactivity  

**TypeScript:** adds type safety to JS  

**UI Frameworks/Libraries:**  
- React (most popular)  
- Angular / Vue  

**CSS Tools:**  
- Preprocessors: Sass (popular)  
- Frameworks: Tailwind CSS (modern), Bootstrap (legacy)  

**Advanced Frontend:**  
- Automated Testing: Jest / Vitest  
- Meta Frameworks: Next.js / Remix (for React)
