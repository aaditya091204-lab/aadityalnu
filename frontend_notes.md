# Frontend Development Notes

## Core Skills:
1. HTML – structure web pages  
2. CSS – style pages and create responsive designs  
3. JavaScript – make pages interactive  
4. TypeScript – adds static typing for robustness  
5. UI Libraries/Frameworks: React, Angular, Vue (React recommended)  

## Additional Skills:
- CSS preprocessors: SASS, LESS (SASS recommended)  
- CSS frameworks: Tailwind (recommended), Bootstrap  
- Automated Testing: Jest or Vitest  
- Meta Frameworks: Next.js (recommended), Remix  

## Visual Validation:
- Mosh uses orange icons for frontend tools  
- React logo shown as blue/white icon  

## Local Context (Fresno):
- Frontend jobs often require React + TypeScript  
- Example: ABC Web Studio hires frontend developers for React projects
